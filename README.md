## 🛠️ General Best Practices

- **Security**: Never commit secrets to version control. Use `.env` files.
- **Performance**: Use appropriate caching and indexing strategies.
- **Code Quality**: Follow linting and formatting standards (Prettier, ESLint, Ruff).
- **Modern Tools**: Prefer modern package managers like `pnpm` or `uv`.
