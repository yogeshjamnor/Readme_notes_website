# Ruby Commands

Common commands for Ruby web projects (Bundler / Rails / Sinatra).

## Ruby + Bundler

```bash
ruby -v
bundle -v
bundle install
bundle update
bundle exec ruby app.rb
```
## Rails

```bash
rails -v
rails new myapp
bin/rails server
bin/rails console
```
> [!TIP]
> Use the **Copy** button in the top-right of code blocks to quickly copy commands to your clipboard.
