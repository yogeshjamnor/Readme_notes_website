# Ruby Debugging

Practical debugging tools for Ruby web apps.

## Basic prints

```rb
pp value
warn \"debug\"
```
## Interactive debugging (common tools)

- `debug` (built-in modern Ruby)
- `byebug` / `pry-byebug` (depending on stack)



> [!TIP]
> Use the **Copy** button in the top-right of code blocks to quickly copy commands to your clipboard.
