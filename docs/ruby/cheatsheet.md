# Ruby Cheatsheet

Quick reference for Ruby syntax commonly used in web development.

## Basics

```rb
name = \"World\"
count = 3
ok = true
```
## Methods

```rb
def add(a, b)
  a + b
end

sum = add(2, 3)
```
## Hashes and arrays

```rb
list = [1, 2, 3]
map = { a: 1, b: 2 }
```
> [!TIP]
> Use the **Copy** button in the top-right of code blocks to quickly copy commands to your clipboard.
