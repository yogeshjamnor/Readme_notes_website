# Install PHP

Notes for setting up PHP for web development.

## What you typically need

- PHP 8+
- Composer
- A web server (Nginx/Apache) or PHP built-in server for local dev
- (Optional) Database driver extensions like `pdo_mysql`, `pdo_pgsql`

## Quick checks

```bash
php -v
composer --version
php -m
```
> [!TIP]
> Use the **Copy** button in the top-right of code blocks to quickly copy commands to your clipboard.
